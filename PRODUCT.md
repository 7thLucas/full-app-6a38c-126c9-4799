# Rapi Laundry — Product Information

## What It Is
A focused daily operational monitoring app for multi-branch laundry businesses in Indonesia. It replaces chaotic WhatsApp group check-ins with a structured daily submission flow, giving the owner one-dashboard visibility across all branches.

## Users
- **Owner / Operator**: Multi-branch laundry owner. Needs full visibility across all locations on a single dashboard without manually chasing each branch via WhatsApp every morning. Wants to be proactive, not reactive.
- **Manajer Cabang (Branch Manager)**: Submits the daily operational report per branch — checklist items, current laundry batch status, supporting photos, and any problems encountered.

## Core MVP Features
1. **Daily Branch Checklist** — Structured per-branch checklist; staff ticks mandatory items before end-of-day submission.
2. **Laundry Status Tracking** — Per-branch status of laundry batches (masuk, dicuci, dikeringkan, siap diambil).
3. **Photo Evidence Upload** — Staff attaches photos to checklist items or problem reports as verifiable proof.
4. **Daily Problem Reports** — Staff logs issues with description and photo; owner sees all open issues surfaced on the dashboard.
5. **Multi-Branch Owner Dashboard** — Single view of all branches' daily completion status, open problem count, and laundry batch overview.

## Positioning
A practical, no-nonsense daily operations tool built specifically for multi-branch laundry businesses in Indonesia. Not a full ERP — just structured daily accountability and visibility, replacing WhatsApp threads with a clean system the whole team can use.

## Tone & Brand
- Clean, practical, trustworthy
- Calm and efficient — not flashy or consumer-oriented
- Indonesian market; primary UI language: Bahasa Indonesia

## Anti-References
- Not a consumer-facing app, not a POS, not an ERP.
- No flashy gimmicks, no gamification, no marketing-heavy chrome.

## Scope Boundaries (MVP)
- In scope: Branch checklist, laundry batch status, photo upload, problem reports, owner dashboard.
- Out of scope (v1): Customer-facing features, POS integration, invoicing, scheduling, payroll.

## Strategic Principles
- Optimize for the daily morning ritual: the owner should understand the state of all branches in seconds.
- Reduce friction for branch managers submitting reports — fast, mobile-friendly, photo-first.
- Make accountability auditable: every submission timestamped, every problem traceable.
